# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python, PostgreSQL, and Power BI**. The project focuses on cleaning raw data, performing SQL-based business analysis, and creating an interactive dashboard to uncover meaningful business insights.

---

## 📌 Project Overview

This project demonstrates the complete Data Analytics workflow, from data preprocessing to visualization. Using customer shopping transaction data, the analysis identifies purchasing patterns, customer segments, and sales trends to support data-driven decision-making.

---

## 📂 Dataset

The dataset contains approximately **3,900 customer records** with information such as:

- Customer Demographics (Age, Gender, Location)
- Product Category
- Purchase Amount
- Review Rating
- Subscription Status
- Shipping Type
- Discounts & Promo Codes
- Previous Purchases
- Payment Method

## 🔄 Project Workflow

### 📌 Data Cleaning & Preprocessing (Python)
- Imported and explored the dataset
- Handled missing values
- Standardized column names
- Performed feature engineering
- Prepared the dataset for SQL analysis

### 📌 SQL Analysis (PostgreSQL)
Performed business analysis using SQL by answering questions related to:
- Customer demographics
- Revenue by category
- Subscription analysis
- Shipping preferences
- Customer purchase behavior
- Review ratings
- Product performance
- Sales trends

### 📌 Interactive Dashboard (Power BI)
Developed an interactive dashboard featuring:
- KPI Cards
- Revenue Analysis
- Sales by Category
- Revenue by Age Group
- Subscription Status Analysis
- Interactive Filters (Slicers)

## 🚀 How to Run

1. Clone this repository.
2. Open `project.ipynb` in Jupyter Notebook.
3. Install the required Python libraries.
4. Load the cleaned dataset into PostgreSQL.
5. Execute the SQL queries.
6. Open the `.pbix` file in Power BI Desktop to explore the dashboard.
